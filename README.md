## Install
### Ubuntu
```bash
git clone https://github.com/ritonun/MetRose.git
cd metrose
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Windows
```bash
git clone https://github.com/ritonun/MetRose.git
cd metrose
python -m venv .venv
.venv\Scripts\Activate
pip install -r requirements.txt
```
